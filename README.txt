Веб-приложение для управления базой данных пород собак с системой разграничения прав доступа (RBAC). Проект разработан в рамках курсовой работы с использованием стека PERN (PostgreSQL, Express, React, Node.js).

Запуск Backend
cd path/to/your/backend
npm install
npm start

Запуск Frontend
cd path/to/your/frontend
npm install
npm start


Проверка БД
psql -U postgres
